## Завдання 2:
Був підібраний префікс довжиною 20 байтiв до повiдомлення «give my friend 2 bitcoins for a pizza», щоб геш-значення утвореного рядка починалося з 32 нулiв. Для цього була написана функція task2(), що працює за методом перебору, щоб знайти потрібі 20 байтів. У виводі є маленькі помилки, що заважають його розумінню — SHA-256 це новий рядок, який показує хеш з цим конкатенованим повідомленням. Для знаходження потрібних 20 байтів знадобилось 2615088202 ітерацій методом перебору поступово збільшуючи 20 байтне число на один кожної ітерації.

## Завдання 8:
Використовуючи власну реалізацію SHA-256 було обчислено відбиток сертифікату kse.ua (сертифікат завантажили з сайту).
